# Транскриптор - преврати фонетическое слово в транскрипцию!

Как пользоваться программой:

1. Загрузить файлы программы
2. Открыть main.py *(удостовериться что на компьютере установлен язык программирования Python)*
3. Запустить main.py
4. В поле ввода ввести нужное фонетическое слово, выделив ударение большой буквой

Например:
> ввод: молокО
>
>вывод:[мълако́]

> ввод: глУхонький
>
> вывод: [глу́хън'к'ьй]
